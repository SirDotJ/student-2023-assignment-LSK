Исходный код программы сохранен в файле src/main/java/org/example/Main.java

Программа разработана на языке программирования Java 17.
Для запуска программы на компьютере необходимо иметь установленную версию Java 17 (а именно jdk-17.0.8).
Инструкции к установке Java 17 предоставлены на официальном сайте Oracle: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html.

К проекту приложен архив с "student-2023-assignment.jar" и "replacement.json" файлами.
"student-2023-assignment.jar" является главным артефактом проекта.

Для запуска программы необходимо открыть терминал в папке где находится распакованные "student-2023-assignment.jar" и "replacement.json" файлы и вписать команду "java -jar имя_jar_файла".

Ввод в программу производится одним из двух способов:
- В качестве аргумента программы;
- Вводом с клавиатуры.

В программу необходимо ввести URL Api для data.json.

После запуска программы в той же папке должен быть создан файл "result.json" с преобразованным текстом из API по правилам, описанных в "replacement.json" файле.
